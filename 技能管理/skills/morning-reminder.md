# 每日晨间日程提醒

## 触发时间
每天 07:00

## 操作步骤

1. **计算今日日期范围**：用 Python 计算今天 00:00 ~ 23:59 的时间戳（禁止手算时间戳）
2. **查询今日日历**：用 `lark-cli calendar +agenda` 查询今日所有日程（默认当天）
3. **查询明日日历**：同上，查询明天的日程
4. **判断凌晨活动**：如果今日有 06:00 前的活动，单独标注「⚠️ 凌晨活动」
5. **推送格式**：简洁，一屏看完

## 推送格式

有活动时：
```
☀️ 今日日程（5月22日 周五）

  09:00  晨会
  10:30  项目评审
  14:00  面试准备

⚠️ 06:00  马拉松（凌晨出发）

📅 明日：14:00 面试
```

无活动时：
```
☀️ 今日日程（5月22日 周五）

今日无安排 🎉

📅 明日：14:00 面试
```

## 注意事项

- 时间左对齐，列表形式，一眼扫完所有时间点
- 凌晨活动（06:00 前）单独标注「⚠️ 」，不归入任何时间段
- 明日预告始终显示（无论今天是否有活动）
- 无活动时只显示无安排提示 + 明日预告，不显示多余内容
