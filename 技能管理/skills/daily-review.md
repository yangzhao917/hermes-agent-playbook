# 每日复盘

## 存放位置

复盘文档必须放在飞书云空间 `hermesAgent/每日复盘/` 文件夹下，不得放云空间根目录。

## 命名格式

`YYYY-MM-DD-复盘总结`

## 文档更新机制

每天只创建/更新一份复盘文档：
1. 先用 Python 计算今天的日期字符串
2. 用 `lark-cli drive +search` 在每日复盘文件夹里搜索当天日期
3. 已有 → 覆盖更新（overwrite）；没有 → 创建新文档

## 分发

微信只发简版（完成情况 + 明日待办 + 待跟进），飞书存全文。

## 数据来源

真实飞书日历 + 任务列表，不编造。活动状态以实际状态描述，不写「圆满结束」等结论性判断。
