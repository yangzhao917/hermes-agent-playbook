# AgentOS 产品简报

## 定位

AgentOS 是一个由微信对话驱动的个人操作系统，帮助杨钊判断什么值得注意，通过飞书和 Hermes skills 执行动作，并沉淀可复盘的事实证据。

## 初始用户

初始用户是杨钊。产品假设用户同时有多条目标、多种个人数据源，并且真正需要的是减少注意力碎片，而不是制造更多任务。

## 核心用户任务

- 早上判断今天真正要抓的 1-2 件事。
- 白天发现 Agent 归类错了，可以直接纠偏。
- 晚上看今天是真推进，还是只是忙。
- 每周基于事实证据决定继续、暂停、委托或归档。
- 长期把阅读、笔记、任务、健康、睡眠、财务和聊天信号接成一个决策系统。

## MVP 范围

本期包含：

- YAML v2 注意力主线治理。
- 晨间提醒、每日复盘、每周复盘。
- 飞书日程、任务、复盘文档。
- 微信读书和 ima 作为知识输入。
- Clawbot 作为微信确认和纠偏入口。

本期不做：

- Hermes core 改造。
- 新数据库。
- Dashboard。
- 健康、睡眠、财务、微信聊天记录全量接入。
- 未确认的自动长期记忆写入。

## 产品指标

- 每天是否生成注意力收口。
- 每日复盘是否区分事实、判断和不确定。
- 周复盘是否包含本周事实和下周取舍。
- 用户纠偏是否被记录，且不会静默变成长期规则。
- 缺数据项是否被明确标注，而不是被模型脑补。
