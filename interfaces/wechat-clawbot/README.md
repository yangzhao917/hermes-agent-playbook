# WeChat Clawbot Interface

WeChat Clawbot is the primary MVP user interface.

## Responsibilities

- Morning attention summary.
- Daily review summary.
- Weekly review summary.
- User correction and confirmation.
- Lightweight diagnostics when the user asks what happened.

## Confirmation Language

Prefer product language over implementation terms:

- "今天真正要推进的"
- "先不用花力气"
- "需要你确认"
- "我不确定的部分"

Avoid exposing internal labels such as `now`, `next`, `later`, or `paused`.
