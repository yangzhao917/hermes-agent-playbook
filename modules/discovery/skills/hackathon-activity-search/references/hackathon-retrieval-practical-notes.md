## 检索路径

公众号标题格式规律：
- `[城市] + 黑客松` = 城市站活动
- `挑战赛 / 联赛 / 大赛` = 正式比赛
- `Meetup / Meetup站` = 线下聚会，可能不是完整黑客松
- `倒计时 / Last Call / 报名即将截止` = 可报名

搜索词参考：
- `城市 + 黑客松 + 2026`
- `黑客松 + 2026 + 报名`
- `AINX / 浦软 / NextStep / AdventureX`（品牌名）
- `6月 + 黑客松 + 2026`
